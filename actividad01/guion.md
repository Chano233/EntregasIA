Guión Sistemas Expertos

El conocimiento en los sistemas expertos es la información y las reglas que un sistema utiliza para resolver problemas o tomar decisiones de forma similar a un especialista humano en un área específica.
Los sistemas expertos se componen principalmente de estos elementos:

Base de conocimiento: Contiene los hechos, conceptos, reglas y procedimientos sobre un dominio específico (por ejemplo, medicina, finanzas o ingeniería).

Motor de inferencia: Analiza la información disponible y aplica las reglas de la base de conocimiento para obtener conclusiones o recomendaciones.

Base de hechos (memoria de trabajo): Almacena los datos del problema que se está resolviendo en ese momento.

Interfaz de usuario: Permite la interacción entre el usuario y el sistema.
Módulo de explicación (en algunos sistemas): Justifica cómo llegó a una determinada conclusión.

La base de conocimientos es el componente central de un sistema experto. Actúa como un depósito digital que almacena los hechos, reglas y heurísticas de un dominio especializado. Su propósito es simular el razonamiento y la toma de decisiones de un profesional humano en esa materia.

Se organiza principalmente en tres elementos:
Hechos: Son la información objetiva, datos concretos y descripciones del dominio (ej. los síntomas específicos de un paciente).
Reglas: Enunciados condicionales que relacionan causas y efectos, generalmente estructurados como "si... entonces" (ej. Si el paciente tiene fiebre alta y tos, entonces la probabilidad de infección es del 80%).
Heurísticas: Estrategias, atajos o reglas empíricas que ayudan a resolver problemas complejos donde no existe una respuesta única.

Un motor de inferencia es el componente de un sistema de Inteligencia Artificial o de software que procesa datos de entrada aplicando reglas lógicas para deducir nueva información, tomar decisiones o generar predicciones. Actúa como el "cerebro" que transforma el conocimiento estático en respuestas útiles.

Componentes Clave 
Funciona en conjunto con una base de conocimientos, utilizando dos elementos principales:

Fase de Emparejamiento: Revisa los datos de entrada y los compara con las reglas predefinidas en el sistema para encontrar coincidencias.
Fase de Ejecución: Aplica las reglas coincidentes para extraer una conclusión o realizar una acción.

Métodos de Razonamiento
Existen dos formas principales en las que el motor opera para resolver problemas:

Encadenamiento hacia adelante: Parte de hechos conocidos y aplica reglas para generar nueva información de forma progresiva. (Ejemplo: Si sabes que "está lloviendo" y "tengo un paraguas", infieres: "no me voy a mojar").
Encadenamiento hacia atrás: Comienza con una hipótesis u objetivo y trabaja hacia atrás para validar qué hechos y reglas son necesarios para confirmarlo.

Tipos de conocimiento en un sistema experto
Conocimiento declarativo: Describe hechos y conceptos.

Conocimiento procedimental: Indica cómo realizar una tarea o resolver un problema.

Conocimiento heurístico: Basado en la experiencia de expertos; utiliza reglas prácticas que ayudan a encontrar soluciones rápidamente.

Metaconocimiento: Es el conocimiento sobre cómo utilizar el propio conocimiento del sistema, es decir, qué reglas aplicar y en qué orden.
